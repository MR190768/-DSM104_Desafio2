# CRUD agencia de viajes

Esta es un aplicación que implementa un CRUD completo de una agencia de viajes con firebase realtimeDatabase, además de registro y login de usuarios con firebase Auth.


# SDK utilizados

Para el manejo de imágenes de utilizo coil por su simplicidad y fácil uso, y cloudinary para guardar imagines en la nube.
